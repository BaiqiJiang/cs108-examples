This is a website for BI 260 Marine Biology students.

To use this website, please create your own profile first.
Create profile now

Note: Unlike editing classes, you can upload a picture of your portrait directly instead of providing a troublesome link. If you don't upload your portrait, we will give you a lovely Boston Terrier by default!

After completing this step, you can view all the phyla, as well as those classes belong to them.
Click here to view all phyla | Click here to view all classes

Although only administrators can add or delete phyla (because this is usually difficult for students to edit), you can add or update any class you have learned.

To update a class, click "update" or "delete" buttons next to each class. To add a class, go into the phylum page and find the "add" button, click it. Note that you must create a profile first before you add or update anything.

Note: Since we want to make sure that the pictures you upload are academic, you must provide a link to the pictures instead of uploading them directly.

If you want to quit, just delete your profile. To do this, go to the profile page and click the "delete" button next to your profile.

If you change your email address and you want to update it, then go to the profile page and click the "update" button next to your profile.

Happy Editing!
